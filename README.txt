Управление:
W - вверх
X - вниз
A - влево
D - вправо
S - вращаться против часовой стрелки
Q - вращаться по часовой стрелке
ПРОБЕЛ - стрелять.

Больший кружочек поглощает меньший, когда оказывается внутри него. 
При этом этого большего размер возрастает. Но если поглощенный кружочек  - это пуля, то
размер, напротив, уменьшается.

исполняемый файл: desktop/build/libs/desktop-1.0.jar
